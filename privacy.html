プライバシーポリシー

最終更新日: 2026-05-27

MeON (以下「本アプリ」) の運営者 (以下「運営者」) は、利用者のプライバシー保護を重要視し、個人情報および利用データの取り扱いについて以下のとおり定めます。

要点だけ知りたい方へ:
  • 写真は端末内に保存され、試着結果生成のとき以外は端末外に出ません。
  • 試着時のみ、暗号化通信で Google Gemini API に送信されますが、Google および当方サーバーで保管はされません。
  • AI の学習には利用されません。
  • 個人を識別できる情報 (氏名・住所・電話番号) は一切取得しません。

1. 取得する情報

  • プロフィール写真 — 利用者が試着のベースとして登録する写真。端末内 (Documents) にのみ保存されます。
  • 服のスクリーンショット / 画像 — 利用者が試着のために選択した画像。端末内および試着履歴として保存されます。
  • 試着結果画像 — AI により生成された画像。端末内の履歴として保存されます。
  • 利用統計 — アプリの起動・機能利用に関する匿名統計 (バージョン情報・OS バージョン等)。個人を特定できない形式で取得します。
  • App Tracking Transparency (ATT) で許可された場合の広告識別子 — 利用者が iOS の設定で許可した場合のみ。

2. 個人情報の利用目的

  • 試着画像の生成および履歴管理
  • 本アプリの機能改善 (匿名統計に基づく)
  • 不具合・問い合わせ対応

3. 第三者への提供

  1. 試着画像の生成のため、利用者の写真 (プロフィール写真および服の画像) は、本アプリ運営者の管理する Cloudflare Worker を経由して Google LLC が提供する Gemini API に送信されます。
  2. 送信は HTTPS による暗号化通信で行われます。
  3. Google LLC は当該データを当該リクエストの処理のためのみに利用し、AI の学習データとしては利用しないことを Google の API 規約上明示しています。詳細は Gemini API の利用規約 をご参照ください。
  4. 運営者の Cloudflare Worker は、リクエストを Gemini に転送した後、レスポンスを利用者の端末へ返却するのみで、写真や生成画像をサーバー側に保管しません。
  5. 上記以外の第三者に対し、利用者の同意なく個人情報を提供することはありません。ただし、法令に基づく場合を除きます。

4. データの保管場所と期間

  • プロフィール写真、服の画像、試着結果はすべて利用者の端末内にのみ保存されます。
  • 運営者は当該データを取得・保管しません。
  • iCloud バックアップが有効になっている場合は、Apple の iCloud に暗号化された形式で含まれます。これは Apple のプライバシーポリシーに従います。
  • 利用者はいつでも履歴削除機能、またはアプリのアンインストールにより、端末内のすべてのデータを削除できます。

5. クッキー・解析ツール

本アプリは現時点でクッキーを利用していません。将来的に Google Analytics for Firebase 等の解析ツールを導入する場合、本ポリシーを更新の上、利用者へ通知します。

6. 広告について

本アプリは将来的に Google AdMob 等の広告ネットワークを利用することがあります。広告配信に伴うトラッキングは、iOS の App Tracking Transparency (ATT) で利用者の許可を取得した上でのみ実施します。許可されない場合でも、本アプリの基本機能は利用できます。

7. アプリ内課金

サブスクリプションおよびポイント購入は、Apple の App Store 経由で行われます。決済情報は Apple が管理し、運営者は決済カード番号などの情報を取得することはありません。

8. 利用者の権利

  • 利用者は、本アプリ内に保存された自身のデータを、いつでも削除することができます。
  • 運営者は利用者のデータをサーバー側に保管していないため、データの開示請求は端末内のデータが対象となります。
  • 個人情報の取り扱いに関するご質問は、お問い合わせ窓口までご連絡ください。

9. 未成年者の利用

13 歳未満の方が本アプリを利用する場合は、保護者の同意のもとでご利用ください。運営者は、未成年者からの本人特定情報を意図的に収集することはありません。

10. ポリシーの変更

本ポリシーは、法令の改正やサービス内容の変更に応じて、必要に応じて改定されます。重要な変更がある場合は、本サポートサイトおよびアプリ内で告知します。

11. お問い合わせ

本ポリシーまたは個人情報の取り扱いに関するご質問は、お問い合わせページ よりご連絡ください。
